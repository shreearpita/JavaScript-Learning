# JavaScript Learning Repository

This repository is a structured and beginner-friendly guide for learning **JavaScript from fundamentals to advanced concepts**.  
It is designed to serve both as a **self learning guide** and a **reference resource** for anyone starting their JavaScript journey.

The repository emphasizes **conceptual clarity**, **clean examples**, and a **progressive learning path**.

---

## Objectives

- Build a strong foundation in JavaScript
- Understand how JavaScript works internally
- Learn JavaScript through organized examples and notes
- Create a reusable reference for future revision
- Help beginners follow a clear, structured roadmap

---

## Repository Structure

The repository is organized into folders based on learning stages:


JavaScript-Learning/

├── 00-pre-requisites/

├── 01-basics/

├── 02-control-flow/

├── 03-functions/

├── 04-objects-arrays/

├── 05-advanced/

├── 06-dom/

└── README.md


---

## Learning Approach

- Concepts are organized in a progressive order
- Examples focus on clarity and simplicity
- Topics move from fundamentals to advanced concepts
- Emphasis is placed on understanding core JavaScript behavior
